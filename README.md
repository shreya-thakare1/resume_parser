# Resume Parser using Python & NLP

This project is a simple yet powerful Resume Parser that extracts important details from PDF resumes automatically. It identifies sections like Skills, Education, and Experience and also detects entities such as names, organizations, locations, and dates using Natural Language Processing.

## Features
- Extracts text from PDF resumes
- Detects resume sections automatically
- Uses NLP for entity recognition
- Beginnerâ€‘friendly and easy to understand
- Clean structured output

## Tech Stack
Python, Regex, pdfplumber, spaCy

## Run
python resume_parser.py

## Use Case
This project is useful for learning NLP concepts and understanding how real resume screening tools work.
